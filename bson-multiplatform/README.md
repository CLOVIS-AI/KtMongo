BSON API for Kotlin, implemented in pure Kotlin with [KotlinX.IO](https://github.com/Kotlin/kotlinx-io) and [KotlinX.Serialization](https://github.com/Kotlin/kotlinx.serialization).

The entry point is [`BsonFactory`][opensavvy.ktmongo.bson.multiplatform.BsonFactory], which provides methods for creating and reading BSON documents.

You can learn more about the Multiplatform implementation and its progress in the [roadmap](https://ktmongo.opensavvy.dev/tutorials/multiplatform/index.html).
